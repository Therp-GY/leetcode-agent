## #15 三数之和（3Sum）
**难度：** 中等

**用户理解方式：**
- 分步骤拆解 + 类比（扑克牌比喻双指针）
- 用"两个人从两边往中间走"代替"双指针"术语

**核心要点：**
1. 先排序
2. 固定第一个数，用两端往里夹找后两个数
3. 谁小谁往里走，不会漏
4. 去重逻辑

## #53 最大子序和
**难度：** 中等

**用户理解方式：**
- 步骤型讲解
- 核心理解：`currentSum` 是"以当前元素结尾的最大连续和"
- 不需要记"动态规划"这个名字

**核心要点：**
1. 遍历数组每个元素
2. `currentSum = max(当前元素, 之前的和 + 当前元素)` — 选"重新开始"或"继续累加"
3. 同时更新 `maxSum` 记录出现过的最大值
4. 核心思想：累加变负数就及时止损

---

## #56 合并区间
**难度：** 中等

**用户理解方式：**
- 步骤型讲解
- Go 排序：`sort.Slice(slice, func(i, j int) bool { return slice[i].Start < slice[j].Start })`

**核心要点：**
1. 按左端点排序
2. 重叠：`下一个左端点 ≤ 当前右端点` → 取右端点最大值合并
3. 不重叠：`下一个左端点 > 当前右端点` → 直接加入结果集
